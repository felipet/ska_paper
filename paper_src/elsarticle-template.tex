\begin{abstract} 
	
	Distributed data adquisition system (DAQ) is a very important component in many scientific and industrial applications. One of the main issues in this field is the synchronization mechanism. All the elements in the network must be synchronized in order to work properly. There are several technologies for the timing synchronization with different accuracy performance such as Network Time Protocol (NTP), Precise Time Protocol (PTP) or White Rabbit (WR). 
	In our contribution, we work in the Square Kilometer Array (SKA) international project framework. The SKA will be the largest radio-telescope of the world and it will be located in two different geographical areas (South-Africa and Australia/New Zealand).
	On each one of those locations, there will be placed hundred of antennas 
	whose individual observations will be combined as if they were acquired 
	by only big one antenna. In order to archive that, all antennas must be synchronized to the same reference very accurately, being required a synchronization performance below 2 ns. This accuracy is not achievable by current standard synchronization technologies. In this context, we propose The WR Zynq Embedded Node (WR-ZEN) platform, a new design based on a Field Programmable Gate Array System-on-Chip (FPGA-SoC), that uses WR technology to reach sub-nanosecond accuracy. This platform has several advantanges compared to the older WR implementations due to the inclusion of an ARM microprocessor and the enhanced clocking circuitry. It enables to add high-level software capabilities in a standard way using conventional operating systems (OS) based on Linux.
	
	The WR-ZEN platform has been tested in different scenarios in order to demostrate the system performance and the behavior in extreme tempeture conditions. It is necessary because SKA's deployment will be built in dessert areas where temperature varies significantly. Finally, the results extracted from the work presented in this contribution reveal that the proposed PPS distribution system meets all SKA's requirements perfectly. Furthermore, our approach is able to compensate synchronization variations caused by temperature change conditions over long fibre links.
\end{abstract}
